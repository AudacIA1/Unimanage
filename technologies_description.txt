Descripción de Tecnologías por Módulo

Este archivo detalla las tecnologías y librerías utilizadas en cada módulo del proyecto UniManage.

---

### Módulo: `accounts`
- **Propósito**: Gestión de la autenticación y perfiles de usuario.
- **Tecnologías**:
    - **Backend**: Django (Python) para la lógica de negocio y vistas.
    - **Frontend**: HTML y Tailwind CSS para las plantillas de inicio de sesión, registro y perfil.
    - **Librerías Adicionales**:
        - `django-crispy-forms`: Para renderizar formularios de manera más limpia.

---

### Módulo: `assets`
- **Propósito**: Gestión de los activos (CRUD), categorías y seguimiento.
- **Tecnologías**:
    - **Backend**: Django (Python) para el manejo de la base de datos y la lógica de negocio.
    - **Frontend**:
        - HTML y Tailwind CSS para las plantillas de listado, creación y edición de activos.
        - JavaScript y `django-autocomplete-light` para la búsqueda predictiva en formularios.
    - **Librerías Adicionales**:
        - `django-mptt`: Para la gestión de categorías jerárquicas de activos.

---

### Módulo: `chatbot`
- **Propósito**: Proporcionar una interfaz conversacional para consultas de usuarios.
- **Tecnologías**:
    - **Backend**:
        - Django (Python) para servir la API del chatbot.
        - `spaCy`: Para el procesamiento de lenguaje natural (NLP) y el entrenamiento del modelo.
    - **Frontend**:
        - JavaScript (`chatbot.js`): Para la lógica de la interfaz del chat, envío de mensajes y renderizado de respuestas.
        - CSS (`chatbot.css`): Para los estilos de la ventana del chat.

---

### Módulo: `dashboard`
- **Propósito**: Visualización de datos y estadísticas clave del sistema.
- **Tecnologías**:
    - **Backend**: Django (Python) para agregar y preparar los datos.
    - **Frontend**:
        - HTML y Tailwind CSS para la estructura del panel.
        - `Chart.js`: Para renderizar gráficos (ej. distribución de activos, préstamos activos).

---

### Módulo: `loans`
- **Propósito**: Gestión de préstamos de activos a usuarios.
- **Tecnologías**:
    - **Backend**: Django (Python) para la lógica de creación, seguimiento y finalización de préstamos.
    - **Frontend**: HTML y Tailwind CSS para los formularios y listados de préstamos.

---

### Módulo: `maintenance`
- **Propósito**: Gestión de las tareas de mantenimiento de los activos.
- **Tecnologías**:
    - **Backend**: Django (Python) para la lógica de registro y seguimiento de mantenimientos.
    - **Frontend**: HTML y Tailwind CSS para los formularios y listados de mantenimiento.

---

### Módulo: `reports`
- **Propósito**: Generación de informes personalizados sobre activos, préstamos y mantenimientos.
- **Tecnologías**:
    - **Backend**: Django (Python) para la recopilación y filtrado de datos.
    - **Frontend**: HTML y Tailwind CSS para la presentación de los informes.
    - **Librerías Adicionales**:
        - `django-weasyprint` (potencial): Para exportar informes a formato PDF.

---

### Módulo: `usermanagement`
- **Propósito**: Administración de usuarios y sus roles en el sistema.
- **Tecnologías**:
    - **Backend**: Django (Python) para el CRUD de usuarios.
    - **Frontend**: HTML y Tailwind CSS para las plantillas de listado, creación y edición de usuarios.
